# "#"
O asterisco serve para criar titulos como você pode ver no texto abaixo. 
Quando mais voce utiliza menor vai ser o titulo 

## _Itálico_
O itálico pode se feito utilizando *(Asterico) ou _(Underscore)

_estou tortinho_

## **Negrito**
Para o negrito você deve colocar ** ou __, em pares 

**Estou cheinho**

### Negrito com itálico
podemos utilizar os dois juntos

**_Estou cheinho que para que eu vou cair_**

## Citações
Para podermos fazer isso temos que utilizar o sinal de >(Maior) como no inicio desse texto

## Listas
Nas lista temos três variações como:

### Lista Ordenada 
A lista ordenada utilizamos (Número)(Ponto) 
Assim temos
1. Ichi
2. Ni
3. San
4. Yon
5. Go

#### Lista Dentro da Lista
Podemos tambem fazer uma lista dentro da lista como:

1. Item 1
2. Item 2
   1. Item 1.1
   2. Item 2.1
3. Item 3

### Lista Não Ordenada
A lista não ordenada utilizamos -(Menos) +(Mais) ou *(Asterisco)

* Item 1
+ Item 2
- Item 3

### Lista de Tarefa
Na lista de tarefa temos uma checkbox para que você possa mostrar o que já foi feito e o que ainda falta fazer

- [ ] Eu estou marcado
- [X] Não estou marcado
